# DevOps Lab

Este repositorio recopila ejercicios prácticos de diversas tecnologías relacionadas con **DevOps**, realizados durante mis prácticas y para mi aprendizaje personal.  
El objetivo es aprender y experimentar con herramientas y conceptos esenciales de DevOps de manera organizada y práctica.

## Tecnologías incluidas

Cada carpeta contiene ejercicios prácticos de la tecnología correspondiente:

- `docker/` → Ejercicios de contenedores y Dockerfiles
- `github-actions/` → Ejercicios de GitHub Actions (workflows, actions, triggers, contextos, variables)
- `kubernetes/` → Ejercicios de despliegue en Kubernetes
- `terraform/` → Ejercicios de infraestructura como código con Terraform
- *(Se pueden agregar más tecnologías según avance el aprendizaje)*

Cada carpeta incluye su propio README.md con explicaciones, objetivos de los ejercicios y archivos necesarios para practicar.

## Objetivo del repositorio

- Aprender y practicar herramientas y conceptos DevOps.
- Mantener los ejercicios organizados por tecnología y subtema.
- Tener un repositorio de referencia personal que también pueda ser útil para otros que quieran aprender.

## Cómo usar este repositorio

1. Navega a la carpeta de la tecnología que quieras practicar.
2. Lee el README de esa carpeta para entender los ejercicios y objetivos.
3. Ejecuta o experimenta con los ejercicios en tu entorno local o repositorios de prueba.
4. Cada ejercicio está diseñado para ser seguro y autónomo.
